Mission Statement

The purpose of this assignment was to exercise collaboration between each developer while creating a website geared towards containerization. Upon visiting the website, a new user should be able to tell the differences between Containerization and Virtualization once they've read through the documentation. Using Docker to create container images is another lesson within itself. Understanding basic commands for the deployment of resources is also another priority. Knowing the tools used within a software development cycle is the best way to keep up with current technological methodologies and best practices.



About

This page its our attempt at explaining the pillars of what makes DevOps. It starts with the definitions of Virtualization and Containerization alongside a comparison of both technologies. For containerization, we focus on Docker containers since they are the primary container technology. We also discus the Linux operating system and some basic commands to get any new user going. We hope you enjoy reading our tutorials too!
